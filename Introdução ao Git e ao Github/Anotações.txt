Guia básico de publicações e commits no github

1## (git config --global user.email "o email desejado") : Para vincular o email do autor no github

2## (git config --global user.name e o nome do usuário) : Para vincular o nome do autor no github

3## (git clone "o link do html") : para clonar o repositório no repositório local.

4## (git status) : Para saber o status atual da minha máquina local.

5## (git add .) : para preparar para commitar.

6## (git commit -m "Aqui o que você quer anotar commitando") : para comitar.

7## (git push origin main) para enviar ao repositório remoto.
